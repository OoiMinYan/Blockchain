---------------------------------------------------
| Initial steps of the blockchain vehicle system  |
---------------------------------------------------

1. Download and install IntelliJ 
	http://www.jetbrains.com/idea/download/index.html#section=windows 
2. Download and install Visual Studio Code 
	https://code.visualstudio.com/
3. Download and install xampp
	https://www.apachefriends.org/download.html
3. Finish install 
4. Download the IntelliJ system.zip & blockchain websites.zip & blockchain_database.sql
5. In IntelliJ 
	Open File or Project --> IntelliJ system.zip
6. For the blockchain websites.zip put at the xampp ---> htdocs
7. In xampp
	Import ---> blockchain_database.sql
	
----------------------------------------------
| Steps to run the blockchain vehicle system  |
----------------------------------------------
1. Open xampp 
2. Click "Start" button for the Apache and MySQL
3. Open the Vehicle Malaysia website
	a. Login as
		admin name : Admin1 
		admin password : admin1pass
		----------------------------
		admin name : Admin2
		admin password : admin2pass
	b. Generate vehicle items
4. Open the Trusted Dealer Malaysia website
	a. Register
	b. Login
	c. Apply new vehicle contract
5. Open intelliJ IDE
	a. run the program
	b. Choose 
		menu selection-2 for add a block
		menu selection-3 for view all block
		menu selection-4 for view one block
		menu selection-6 for add all block (generate installment records)
		menu selection-7 for sync contract
		menu selection-0 for quit and generate pdf
6. Continue the Trusted Dealer Malaysia website
	For-Sale page will show can-sale vehicle table
	a. Apply second-hand vehicle contract
